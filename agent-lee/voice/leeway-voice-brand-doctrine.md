# LeeWay Voice Brand Doctrine

## LeeWay Voice Brand Doctrine

LeeWay Voice is the governed voice command layer for Agent Lee.

It is not generic browser speech.
It is not a raw transcript pipe.
It is not a hidden automation trigger.
It is not a fake assistant voice.
It is not a status logger.

LeeWay Voice means:
- the owner's spoken intent is received with respect
- the system classifies the intent truthfully
- voice input becomes a governed request, not a blind command
- agents may assist only under LeeWay law
- mutating actions require proposal and approval
- degraded capability is named honestly
- the owner always knows what was heard, what was understood, what can happen next, and what cannot happen

Brand promise:
LeeWay Voice helps the owner speak ideas into motion without losing control.

Core line:
The owner speaks. LeeWay classifies. Agents assist. Law governs.

LeeWay does not hide degraded capability. It names it, protects the owner, and keeps a safe path open.
