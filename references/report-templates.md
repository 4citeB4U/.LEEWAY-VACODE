# LeeWay Report Templates

LeeWay applications should generate clear reports for every major governance surface.

## Required reports
- `leeway_compliance_report.md`
- `leeway_owner_acceptance_report.md`
- `leeway_owner_education_qa_report.md`
- `leeway_selected_area_agent_runtime_report.md`
- `leeway_agent_skill_runtime_report.md`
- `leeway_agent_capability_report.md`
- `leeway_security_architecture_report.md`
- `leeway_data_classification_report.md`
- `leeway_mcp_registry_report.md`
- `leeway_runtime_truth_report.md`
- `leeway_exceptions_register.md`

## Report requirements
- clearly state PASS / PARTIAL / FAIL
- state what was tested, what was inspected, and what was actually run
- include exact commands executed
- include evidence references and caveats
- distinguish claims from verified proof
- expose missing standards source or broken connection state
