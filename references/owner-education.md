# LeeWay Owner Education

Owner education is a required part of every owner-facing LeeWay application.

## Required content
- printable manual with clear sections and table of contents
- Markdown export that preserves TOC and troubleshooting
- contextual help triggers attached to UI controls
- onboarding start/skip controls
- real onboarding target selectors for owner-facing features
- troubleshooting section with escalation guidance
- owner-readiness QA in the workflow
- traceability metadata for all help and manual entries

## Enforcement
- do not treat registry presence as sufficient evidence of owner education
- help must be surfaced in the UI, not only stored in backend registries
- onboarding must reference actual controls and user flows
- escalation guidance must include next steps and contact/agent routes
- owner education must be visible, printable, traceable, and usable
