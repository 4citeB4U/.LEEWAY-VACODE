# LeeWay Owner Acceptance QA

## Purpose
Ensure owners can use, govern, and approve the system independently.

## Must verify
- theme editing workflow
- public section editing workflow
- paired AdminOS workflow and public/admin projection
- agent proposal creation and approval
- agent status and blocked capability visibility
- MCP status and connection state
- growth lane classification (Green/Yellow/Red)
- product control and owner authority surfaces
- runtime authority and publish controls
- manual/help/onboarding/troubleshooting support
- AdminOS dashboard, security center, and audit surfaces

- `leeway_owner_acceptance_report.md`
- `leeway_owner_education_qa_report.md`
