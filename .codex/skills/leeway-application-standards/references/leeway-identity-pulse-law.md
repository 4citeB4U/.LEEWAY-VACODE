# LeeWay Identity Pulse Reference

LeeWay Identity Pulse is the living heartbeat of the system.

Every governed object must be traceable by identity, origin, lineage, trust, ownership, hash, verification path, and receipt before it may act.

Required practical rule:

- if the object is born in LeeWay, mark it as `LEEWAY_BORN`
- if the object entered from outside LeeWay, mark it as imported, downloaded, restored, uploaded, tool-generated, quarantined, or unknown
- if the object has no identity, it has no authority
- if the object has no origin, it has no trust
- if the object has no receipt, it has no truth
